# 🌍 Länder-Webanwendung

Eine kleine Webanwendung mit PHP, die Länderinformationen über eine externe API lädt und darstellt.

---

## 📌 Beschreibung

Dieses Projekt zeigt eine Übersicht aller Länder inklusive Flaggen.
Zusätzlich bietet die Anwendung:

* eine Detailseite für jedes Land
* eine Suchfunktion zur gezielten Abfrage von Ländern

Die Daten werden dynamisch über die REST Countries API geladen.

---

## 🚀 Features

* 🌐 Anzeige aller Länder mit Flaggen
* 🔗 Klickbare Länderkarten mit Weiterleitung zur Detailseite
* 📄 Detailansicht mit:

  * Name
  * Hauptstadt
  * Region
  * Bevölkerung
  * Flagge
* 🔍 Suchfunktion für Länder
* 🧭 Navigation zwischen Übersicht, Detailseite und Suche

---

## 🛠 Technologien

* HTML
* CSS
* PHP
* REST API (https://restcountries.com)

---

## ▶️ Installation & Nutzung

1. Repository klonen:

   ```bash
   git clone <https://github.com/Emmo65/Website-L-nder-API>
   ```

2. Projekt in den `htdocs`-Ordner von XAMPP legen

3. XAMPP starten (Apache aktivieren)

4. Im Browser öffnen:

   ```text
   http://localhost/<neu Länderseite>/index.php
   ```
